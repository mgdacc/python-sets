Cheat Sheet de Python

Conjuntos (Sets)

1 / 15

¿Qué es un Conjunto?

Una colección desordenada de elementos únicos.

2 / 15

Crear un Conjunto

Se define con llaves {} o con la función set().

# Con llaves
numeros = {1, 2, 3, 4}

# Con la función set()
letras = set("palabra") # {'p', 'a', 'l', 'b', 'r'}
3 / 15

Crear un Conjunto Vacío

¡Importante! Se debe usar set(), ya que {} crea un diccionario vacío.

conjunto_vacio = set()
diccionario_vacio = {}
4 / 15

No Permiten Duplicados

Los elementos duplicados se eliminan automáticamente.

lista = [1, 2, 2, 3, 1, 4]
conjunto = set(lista)
print(conjunto) # {1, 2, 3, 4}
5 / 15

Son Desordenados

No se puede acceder a los elementos por un índice.

mi_conjunto = {1, 2, 3}
# Esto dará un error (TypeError)
print(mi_conjunto[0])
6 / 15

Añadir Elementos: .add()

Añade un único elemento al conjunto.

numeros = {1, 2, 3}
numeros.add(4)
print(numeros) # {1, 2, 3, 4}
7 / 15

Añadir Elementos: .update()

Añade múltiples elementos de otro iterable.

numeros = {1, 2}
numeros.update([3, 4, 5])
print(numeros) # {1, 2, 3, 4, 5}
8 / 15

Eliminar Elementos: .remove()

Elimina un elemento. Da error si el elemento no existe.

numeros = {1, 2, 3}
numeros.remove(2)
# numeros.remove(99) # Daría un error (KeyError)
9 / 15

Eliminar Elementos: .discard()

Elimina un elemento. No da error si el elemento no existe.

numeros = {1, 2, 3}
numeros.discard(3)
numeros.discard(99) # No pasa nada
print(numeros) # {1, 2}
10 / 15

Operación: Unión (|)

Combina todos los elementos de dos conjuntos.

A = {1, 2, 3}
B = {3, 4, 5}
print(A | B) # {1, 2, 3, 4, 5}
11 / 15

Operación: Intersección (&)

Obtiene los elementos que están en ambos conjuntos.

A = {1, 2, 3}
B = {3, 4, 5}
print(A & B) # {3}
12 / 15

Operación: Diferencia (-)

Obtiene los elementos del primer conjunto que no están en el segundo.

A = {1, 2, 3}
B = {3, 4, 5}
print(A - B) # {1, 2}
13 / 15

Operador in

Verifica si un elemento existe en el conjunto. Es una operación muy rápida.

numeros = {1, 2, 3, 4, 5}
if 3 in numeros:
    print("El 3 está en el conjunto.")
14 / 15

Recorrer un Conjunto

Se usa un ciclo `for`. El orden de los elementos no está garantizado.

for numero in {1, 2, 3}:
    print(numero)
